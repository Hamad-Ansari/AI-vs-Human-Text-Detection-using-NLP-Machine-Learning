# 🔍 AI vs Human Text Detector — Streamlit App

Detects whether text was written by a human or generated by AI,
using TF-IDF + classical ML and a Bidirectional LSTM model.

---

## 📁 Required Files

Place all of these in the **same folder** as `app.py`:

```
app.py
best_ml_model.pkl
tfidf_vectorizer.pkl
lstm_model.h5
lstm_tokenizer.pkl
```

---

## ⚙️ Setup

```bash
# 1. Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

# 2. Install dependencies
pip install -r requirements.txt
```

---

## 🚀 Run the App

```bash
streamlit run app.py
```

The app will open in your browser at **http://localhost:8501**

---

## 🎯 Features

| Tab | What it does |
|-----|--------------|
| **Single Text** | Paste one piece of text and get an instant prediction with confidence score |
| **Bulk Paste** | Paste multiple texts (one per line) and analyze them all at once |
| **Upload CSV** | Upload a CSV file, choose the text column, and run batch predictions |

### Model Options (Sidebar)
- **Ensemble** — Averages ML + LSTM probabilities (most accurate)
- **ML Model** — Fast TF-IDF + classical ML prediction
- **LSTM** — Deep learning Bidirectional LSTM prediction

---

## 🧠 How It Works

1. **Text preprocessing** — lowercasing, punctuation removal, lemmatization, stop-word removal (mirrors the training notebook)
2. **TF-IDF path** — vectorizes the cleaned text and runs through the saved best ML classifier
3. **LSTM path** — tokenizes → pads to 200 tokens → runs through the Bidirectional LSTM
4. **Ensemble** — averages both probabilities for a final decision
